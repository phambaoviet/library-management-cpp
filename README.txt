--------------------------------------------------------
   ĐỒ ÁN 1 - PHẦN MỀM QUẢN LÝ THƯ VIỆN (C++ - Console)
--------------------------------------------------------

📌 CÁCH BIÊN DỊCH VÀ CHẠY CHƯƠNG TRÌNH TRÊN TERMINAL:

1. Biên dịch chương trình:

    g++ main.cpp phieuMuonSach.cpp phieuTraSach.cpp quanlysach.cpp docgia.cpp thongke.cpp data.cpp -o program

2. Sau khi biên dịch thành công, chạy chương trình bằng lệnh:

   ./program

3. Các chức năng chính
   - Quản lý độc giả 
   - Quản lý sách
   - Lập phiếu mượn và trả sách
   - Thống kê số liệu trong thư viện
---------------------------------
📁 Danh sách các file mã nguồn:
---------------------------------
- main.cpp
- docgia.cpp + docgia.h
- quanlysach.cpp + quanlysach.h
- phieuMuonSach.cpp + muonSach.h
- phieuTraSach.cpp + traSach.h + thongKe.h

💡 Lưu ý:
- Dữ liệu KHÔNG được lưu xuống file, chỉ chạy trong RAM.
- Sử dụng các kiểu dữ liệu cơ bản, mảng 1 chiều, không dùng struct.
